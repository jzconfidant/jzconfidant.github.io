ESP8266开发笔记

发表于 2017-09-07 | 分类于 嵌入式 | | 阅读次数

1.准备工作

1.1 硬件开发板

我这边选择的是NodeMCU开发板，基于安信可的ESP-12F模块，一根USB线解决供电、烧录、打印功能。

1.2 编译环境配置

编译环境的配置方式有两种：

  1. 编译环境可以到乐鑫官网去下载linux编译镜像。
  2. 只下载编译器，自己去配置开发环境（linux平台），可参考此处。

2. 编译

ESP8266提供了两个版本的SDK，分别是ESP8266_NONOS_SDK和ESP8266_RTOS_SDK，这里我选择了NONOS_SDK，因为其比较简单，且网上基于它开发的资源比较多。RTOS_SDK是后来才提供的，相对资料少点。
